About BreedingAIDB

BreedingAIDB will promote application of ML in breeding practices. This resource provides a valuable foundation for the application of machine learning in G2P prediction and breeding practices.

Introduction

BreedingAIDB provide breeders and machine learning experts with easily accessible genotype-to-phenotype pair data of crops (the Release 1.0 includes rice, soybean, and maize), and G2P prediction tools.

Crops in BreedingAIDB

Rice

129, 449 rice genome-to-phenotype pairs available here.

Maize

16, 284 maize genome-to-phenotype pairs available here.

Type of G2P Data

GVCF-to-phenotype pairs

  • Unstructured data
  • GVCF files cover all loci in the genome

VCF-to-phenotype pairs

  • Unstructured data
  • VCF files contain high-quality SNP information

gsctool-to-phentype pairs

  • Structured data
  • The gsctool features are generated by GSCtool

ML Tools

Feature Extraction

The module utilizes GSCtool to extract essential genomic features.

Phenotype Prediction

The module offers predictions for grain length and width of rice, which can be easily done by uploading the feature files extracted by GSCtool

ML Project

The module integrates Optuna and lightGBM to allow users to customize and optimize ML models for their specific research needs.
